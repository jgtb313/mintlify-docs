---
title: "List Memberships"
api: "GET /organizations/{organizationId}/memberships"
description: "List memberships."
---

Returns a list of memberships.

<Info>Requires `memberships:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Query Parameters

<ParamField query="previousCursor" type="string">
  Cursor for the previous page.
</ParamField>

<ParamField query="nextCursor" type="string">
  Cursor for the next page.
</ParamField>

<ParamField query="limit" type="integer">
  Number of results per page. Default: 20.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of membership objects.
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata.
