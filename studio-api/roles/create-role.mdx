---
title: "Create Role"
api: "POST /organizations/{organizationId}/roles"
description: "Create a role."
---

Creates a new role.

<Info>Requires `roles:write` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Request Body

<ParamField body="slug" type="string" required>
  Unique role identifier within the organization.
</ParamField>

<ParamField body="name" type="string" required>
  Display name for the role.
</ParamField>

<ParamField body="description" type="string">
  Optional description.
</ParamField>

<ParamField body="permissions" type="string[]" required>
  Array of permission slugs (e.g., `["users:read", "roles:read"]`).
</ParamField>

## Response

<ResponseField name="roleId" type="string">
  The role ID.
</ResponseField>

<ResponseField name="organizationId" type="string">
  The organization UUID. Present only for organization roles.
</ResponseField>

<ResponseField name="slug" type="string">
  Unique role identifier.
</ResponseField>

<ResponseField name="name" type="string">
  Display name.
</ResponseField>

<ResponseField name="description" type="string">
  Role description.
</ResponseField>

<ResponseField name="type" type="string">
  Role type (`ENVIRONMENT_ROLE` or `ORGANIZATION_ROLE`).
</ResponseField>

<ResponseField name="permissions" type="string[]">
  Array of permission slugs.
</ResponseField>

<ResponseField name="inherited" type="boolean">
  Whether the role is inherited from the environment.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 creation timestamp.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 last-update timestamp.
</ResponseField>

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |

| `403` | `forbidden` | Insufficient permissions |
| `404` | `organization.not_found` | Organization not found |
| `409` | `role.slug_already_exists` | Role slug is already in use |
| `404` | `permission.not_found` | One or more permissions do not exist |
| `500` | `internal_server_error` | An unexpected error occurred |
