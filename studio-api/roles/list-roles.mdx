---
title: "List Roles"
api: "GET /organizations/{organizationId}/roles"
description: "List roles."
---

Returns a list of roles.

<Info>Requires `roles:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Response

The response is an array of role objects.
