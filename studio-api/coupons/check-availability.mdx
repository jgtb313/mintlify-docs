---
title: "Check Coupon Availability"
api: "GET /coupons/{code}/availability"
description: "Check coupon availability."
---

Checks coupon availability.

<Info>Requires `coupon:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="code" type="string" required>
  The coupon code to check.
</ParamField>

## Response

<ResponseField name="coupon" type="object">
  The coupon object.
</ResponseField>

<ResponseField name="meta" type="object">
  Availability metadata.
  <Expandable title="meta properties">
    <ResponseField name="available" type="boolean">
      Whether the coupon can be redeemed.
    </ResponseField>
  </Expandable>
</ResponseField>
