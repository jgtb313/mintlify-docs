---
title: "List Coupon Usages"
api: "GET /organizations/{organizationId}/coupons"
description: "List coupon usages."
---

Returns a list of coupon usages.

<Info>Requires `coupon:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Page number.
</ParamField>

<ParamField query="limit" type="integer">
  Results per page.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of coupon usage objects.
</ResponseField>
