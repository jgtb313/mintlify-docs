---
title: "List Invitations"
api: "GET /organizations/{organizationId}/invitations"
description: "List invitations."
---

Returns a list of invitations.

<Info>Requires `invitations:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Query Parameters

<ParamField query="previousCursor" type="string">
  Cursor for the previous page.
</ParamField>

<ParamField query="nextCursor" type="string">
  Cursor for the next page.
</ParamField>

<ParamField query="limit" type="integer">
  Number of results per page.
</ParamField>

<ParamField query="email" type="string">
  Filter by email address.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of invitation objects.
