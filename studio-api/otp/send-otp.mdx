---
title: "Send OTP"
api: "POST /otps"
description: "Send a one-time password."
authMethod: "none"
---

Sends a one-time password to the recipient.

## Request Body

<ParamField body="recipient" type="string" required>
  The recipient's email address or phone number.
</ParamField>

<ParamField body="context" type="string" required>
  The OTP context. One of: `EMAIL_VERIFICATION`, `MAGIC_AUTH`, `RESET_PASSWORD`.
</ParamField>

## Response

<ResponseField name="otpId" type="string">
  Unique identifier for the OTP request. Used in subsequent validation.
</ResponseField>

## Contexts

| Context | Usage |
|---------|-------|
| `EMAIL_VERIFICATION` | Verify email after sign-up (also used for resend) |
| `MAGIC_AUTH` | Passwordless login |
| `RESET_PASSWORD` | Password reset flow |

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |
| `403` | `auth.method_not_allowed` | Authentication method not allowed by organization |
| `409` | `auth.email_already_verified` | Email has already been verified |
| `500` | `internal_server_error` | An unexpected error occurred |
