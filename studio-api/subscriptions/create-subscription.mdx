---
title: "Create Subscription"
api: "POST /organizations/{organizationId}/subscriptions"
description: "Create a subscription."
---

Creates a new subscription.

<Info>Requires `subscription:write` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Request Body

<ParamField body="planId" type="string" required>
  The plan ID to subscribe to.
</ParamField>

<ParamField body="planIntervalId" type="string" required>
  The plan interval ID (determines billing frequency and amount).
</ParamField>

<ParamField body="paymentMethodId" type="string" required>
  The payment method ID to charge.
</ParamField>

<ParamField body="coupon" type="object">
  Optional coupon to apply at subscription creation.
  <Expandable title="coupon properties">
    <ParamField body="code" type="string" required>
      The coupon code.
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="updatedBy" type="string">
  User who last updated the subscription.

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |
| `401` | `unauthorized` | Missing or invalid authentication |
| `403` | `forbidden` | Insufficient permissions |
| `404` | `organization.not_found` | Organization not found |
| `409` | `subscription.active_subscription_exists` | Organization already has an active subscription |
| `409` | `coupon.not_available` | Coupon is invalid, expired, or depleted |
| `500` | `internal_server_error` | An unexpected error occurred |
