---
title: "Switch Organization"
api: "POST /profile/switch-organization"
description: "Switch the active organization."
---

Switches the active organization context.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Request Body

<ParamField body="organizationId" type="string" required>
  The UUID of the organization to switch to.
</ParamField>

## Response

<ResponseField name="accessToken" type="string">
  JWT access token scoped to the selected organization.
</ResponseField>

<ResponseField name="refreshToken" type="string">
  New refresh token.
</ResponseField>

<ResponseField name="expiresIn" type="integer">
  Access token expiration in seconds.
</ResponseField>
