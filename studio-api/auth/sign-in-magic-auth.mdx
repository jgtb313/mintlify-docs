---
title: "Sign In Magic Auth"
api: "POST /auth/sign-in/magic-auth"
description: "Complete passwordless sign-in."
authMethod: "none"
---

Completes the passwordless sign-in.

## Request Body

<ParamField body="otpId" type="string" required>
  The OTP request identifier.
</ParamField>

<ParamField body="code" type="string" required>
  The verified OTP code.
</ParamField>

## Response

<ResponseField name="accessToken" type="string">
  JWT access token.
</ResponseField>

<ResponseField name="refreshToken" type="string">
  Refresh token.
</ResponseField>

<ResponseField name="expiresIn" type="integer">
  Access token expiration in seconds.
</ResponseField>
