---
title: "Reset Password"
api: "POST /auth/reset-password"
description: "Reset user password."
authMethod: "none"
---

Resets the user's password.

## Request Body

<ParamField body="otpId" type="string" required>
  The OTP request identifier.
</ParamField>

<ParamField body="code" type="string" required>
  The validated OTP code.
</ParamField>

<ParamField body="password" type="string" required>
  The new password.
</ParamField>

## Response

<ResponseExample>
```json 204
// No content — password reset successfully
```

```json 400
{
  "statusCode": 400,
  "message": "auth.invalid_token",
  "errors": [
    {
      "message": "required",
      "path": "field"
    }
  ]
}
```

```json 500
{
  "statusCode": 500,
  "message": "Internal Server Error"
}
```
</ResponseExample>

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |
| `400` | `auth.invalid_token` | Reset token is invalid or expired |
| `400` | `auth.password_too_weak` | Password does not meet strength requirements |
| `500` | `internal_server_error` | An unexpected error occurred |
