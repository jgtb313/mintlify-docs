---
title: "List Invoices"
api: "GET /organizations/{organizationId}/invoices"
description: "List invoices."
---

Returns a list of invoices.

<Info>Requires `invoice:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer">
  Results per page.
</ParamField>

<ParamField query="previousCursor" type="string">
  Cursor to fetch the previous page.
</ParamField>

<ParamField query="nextCursor" type="string">
  Cursor to fetch the next page.
</ParamField>

<ParamField query="paymentMethod" type="string">
  Filter by payment method.
</ParamField>

<ParamField query="status" type="string">
  Filter by invoice status.
</ParamField>

<ParamField query="type" type="string">
  Filter by invoice type.
</ParamField>

<ParamField query="sort" type="string">
  Sort by `status`, `type`, `dueDate`, or `createdAt`.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of invoice objects.

  Pagination metadata.
