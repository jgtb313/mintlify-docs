---
title: "Reissue Payment"
api: "POST /organizations/{organizationId}/invoices/{invoiceId}/reissue-payment"
description: "Reissue an invoice payment."
---

Reissues a payment for an invoice.

<Info>Requires `invoice:reissue` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

<ParamField path="invoiceId" type="string" required>
  The invoice UUID.
</ParamField>

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |
| `401` | `unauthorized` | Missing or invalid authentication |
| `403` | `forbidden` | Insufficient permissions |
| `404` | `organization.not_found` | Organization not found |
| `404` | `invoice.not_found` | Invoice not found |
| `422` | `invoice.not_expired` | Invoice payment has not expired |
| `500` | `internal_server_error` | An unexpected error occurred |
