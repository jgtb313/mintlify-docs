---
title: "List Voucher Usages"
api: "GET /organizations/{organizationId}/vouchers/{voucherId}/usages"
description: "List voucher usages."
---

Returns a list of voucher usages.

<Info>Requires `voucher:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>
  The organization UUID.
</ParamField>

<ParamField path="voucherId" type="string" required>
  The voucher UUID.
</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Page number.
</ParamField>

<ParamField query="limit" type="integer">
  Results per page.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of voucher usage objects.
</ResponseField>
