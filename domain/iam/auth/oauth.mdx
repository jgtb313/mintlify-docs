## Supported Providers

| Provider | Identifier | Description |
|----------|------------|-------------|
| Google | `GoogleOAuth` | Google Workspace and personal accounts |
| Microsoft | `MicrosoftOAuth` | Microsoft 365 and personal accounts |

## Requirements

- User must have an existing account and active membership
- OAuth identity must be linked in WorkOS
- Multi-org users need organization selection after OAuth
