---
title: "cost.requested"
description: "Published when a usage cost is recorded for an organization."
---

## Event Details

| Property | Value |
|----------|-------|
| **Type** | `cost.requested` |

## Payload Schema

<ResponseField name="costId" type="string (UUID v7)" required>
  Unique identifier of the cost record.
</ResponseField>

<ResponseField name="organizationId" type="string (UUID v7)" required>
  FK to Organization.
</ResponseField>

<ResponseField name="feeId" type="string (UUID v7)" required>
  FK to Fee.
</ResponseField>

<ResponseField name="tokenId" type="string (UUID v7)" required>
  FK to Token.
</ResponseField>

<ResponseField name="costBatchId" type="string (UUID v7)">
  FK to CostBatch.
</ResponseField>

<ResponseField name="costBatchWindowRef" type="string" required>
  Unique window reference for batch grouping.
</ResponseField>

<ResponseField name="quantity" type="integer" required>
  Units consumed.
</ResponseField>

<ResponseField name="unitAmount" type="decimal" required>
  Price per unit.
</ResponseField>

<ResponseField name="unitAmountCurrency" type="string" required>
  Token's currency (`USD`, `BRL`, `EUR`).
</ResponseField>

<ResponseField name="subtotalAmount" type="decimal" required>
  quantity × unitAmount.
</ResponseField>

<ResponseField name="exchangeRate" type="decimal" required>
  Conversion rate applied.
</ResponseField>

<ResponseField name="exchangeRateProvider" type="string" required>
  Exchange rate provider.
</ResponseField>

<ResponseField name="totalAmount" type="decimal" required>
  Final amount in organization currency.
</ResponseField>

<ResponseField name="totalAmountCurrency" type="string" required>
  Organization's currency (`USD`, `BRL`, `EUR`).
</ResponseField>

<ResponseField name="createdAt" type="string (ISO 8601)" required>
  Creation timestamp.
</ResponseField>

<ResponseField name="updatedAt" type="string (ISO 8601)" required>
  Last update timestamp.
</ResponseField>
