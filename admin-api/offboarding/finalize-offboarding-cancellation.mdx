---
title: "Finalize Offboarding Cancellation"
api: "POST /organizations/{organizationId}/offboarding-requests/{offboardingRequestId}/finalize"
description: "Confirm the organization cancellation and finalize the offboarding process."
---

<Info>Requires `offboarding:finalize` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>Organization ID (UUIDv7).</ParamField>
<ParamField path="offboardingRequestId" type="string" required>Offboarding request ID (UUIDv7).</ParamField>

## Request Body

<ParamField body="cancelledReason" type="string">The reason for finalizing the cancellation.</ParamField>
