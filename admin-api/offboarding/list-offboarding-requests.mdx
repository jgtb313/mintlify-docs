---
title: "List Offboarding Requests"
api: "GET /organizations/{organizationId}/offboarding-requests"
description: "List offboarding requests for an organization."
---

Returns a paginated list of offboarding requests for the organization.

<Info>Requires `offboarding:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="organizationId" type="string" required>Organization ID (UUIDv7).</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Page number.
</ParamField>

<ParamField query="limit" type="integer">
  Results per page.
</ParamField>
