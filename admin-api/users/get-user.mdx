---
title: "Get User"
api: "GET /users/{userId}"
description: "Retrieve a user by its ID."
---

<Info>Requires `users:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="userId" type="string" required>
  The user ID.
</ParamField>
