---
title: "Delete User"
api: "DELETE /users/{userId}"
description: "Delete a user."
---

Permanently deletes a user from both the local database and the identity provider.

<Info>Requires `users:delete` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="userId" type="string" required>
  The user ID.
</ParamField>
