---
title: "Create Token"
api: "POST /tokens"
description: "Create a token."
---
<Info>Requires `token:write` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>Token name. Max 255 chars.</ParamField>
<ParamField body="description" type="string" required>Description.</ParamField>
<ParamField body="type" type="string" required>
  Token type. One of: `AUDIO_TO_TEXT`, `IMAGE_TO_TEXT`, `MEMORY`, `PLANNING`, `RESPONSE`, `RETRIEVAL`, `SUMMARY`, `TOOLS`, `TRANSCRIPTION`, `WEBSCRAPING`.
</ParamField>
<ParamField body="value" type="string" required>Decimal value per unit.</ParamField>
<ParamField body="currency" type="string" required>One of: `USD`, `BRL`, `EUR`.</ParamField>

## Response

<ResponseField name="tokenId" type="string">Token UUID.</ResponseField>
<ResponseField name="name" type="string">Token name.</ResponseField>
<ResponseField name="description" type="string">Token description.</ResponseField>
<ResponseField name="type" type="string">Token type.</ResponseField>
<ResponseField name="value" type="string">Decimal value per unit.</ResponseField>
<ResponseField name="currency" type="string">Currency code.</ResponseField>
<ResponseField name="status" type="string">Token status.</ResponseField>
<ResponseField name="createdBy" type="string">Creator identifier.</ResponseField>
<ResponseField name="createdAt" type="string">Creation timestamp.</ResponseField>
<ResponseField name="updatedBy" type="string">Last updater identifier.</ResponseField>
<ResponseField name="updatedAt" type="string">Last update timestamp.</ResponseField>

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |
| `401` | `unauthorized` | Missing or invalid authentication |
| `403` | `forbidden` | Insufficient permissions |
| `409` | `token.name_type_currency_already_exists` | Token with same name, type, and currency already exists |
| `500` | `internal_server_error` | An unexpected error occurred |
