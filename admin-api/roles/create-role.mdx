---
title: "Create Role"
api: "POST /roles"
description: "Create a role."
---

<Info>Requires `roles:write` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Request Body

<ParamField body="organizationId" type="string">
  The organization UUID.
</ParamField>

<ParamField body="slug" type="string" required>Unique role slug.</ParamField>
<ParamField body="name" type="string" required>Display name.</ParamField>
<ParamField body="description" type="string">Optional description.</ParamField>
<ParamField body="permissions" type="string[]" required>Array of permission slugs.</ParamField>

## Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `validation_error` | Invalid input or missing required fields |
| `401` | `unauthorized` | Missing or invalid authentication |
| `403` | `forbidden` | Insufficient permissions |
| `409` | `role.slug_already_exists` | Role slug is already in use |
| `404` | `permission.not_found` | One or more permissions do not exist |
| `500` | `internal_server_error` | An unexpected error occurred |
