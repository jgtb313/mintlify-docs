---
title: "Update Role"
api: "PATCH /roles/{slug}"
description: "Update a role."
---

<Info>Requires `roles:write` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>Role slug.</ParamField>

## Query Parameters

<ParamField query="organizationId" type="string">
  The organization UUID.
</ParamField>

## Request Body

<ParamField body="name" type="string">Updated name.</ParamField>
<ParamField body="description" type="string">Updated description.</ParamField>
<ParamField body="permissions" type="string[]">Updated permissions.</ParamField>
