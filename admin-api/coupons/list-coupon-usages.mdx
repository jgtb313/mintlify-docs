---
title: "List Coupon Usages"
api: "GET /coupons/{couponId}/usages"
description: "List coupon usages."
---
<Info>Requires `coupon:read` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="couponId" type="string" required>Coupon UUID.</ParamField>

## Query Parameters

<ParamField query="page" type="integer">Page number.</ParamField>
<ParamField query="limit" type="integer">Results per page.</ParamField>
