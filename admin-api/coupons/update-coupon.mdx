---
title: "Update Coupon"
api: "PATCH /coupons/{couponId}"
description: "Update a coupon."
---
<Info>Requires `coupon:write` permission.</Info>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token.
</ParamField>

## Path Parameters

<ParamField path="couponId" type="string" required>Coupon UUID.</ParamField>

## Request Body

<ParamField body="name" type="string">Updated name.</ParamField>
